# forget-me-not

Forgot your laundry in the washer? Not anymore.

A standalone, localStorage-backed PWA for recurring task management with urgency tracking, follow-up chains, and overdue alerts.

## Run

```
npm install
npm run dev
```

## Build

```
npm run build
```

Static output lands in `dist/`. Deploy anywhere that serves static files.

## Data

All data lives in localStorage. Export/import JSON from Settings. Sync endpoint config available for self-hosted backends.
